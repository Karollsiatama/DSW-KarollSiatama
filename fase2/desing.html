Que es una Etiqueta de HTML

según xavir idevik (2021) El hipertexto (HTML) es un lenguaje informático que forma la mayoría de las páginas web y plataformas online. HTML no se considera un lenguaje de programación, ya que no puede crear una funcionalidad dinámica. Sin embargo, los usuarios web pueden crear y estructurar secciones, párrafos y enlaces usando elementos, etiquetas y atributos. HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero.

¿Que es un atributo? : son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo de elemento de HTML. Un atributo tampoco modifica la funcionalidad por defecto de un tipo correctamente sin ellos. En sintaxis HTML, un atributo se añade a una etiqueta de inicio de HTML.

¿Que es un valor ? : Son las propiedades que se le pueden asignar a los elementos. Por ejemplo, a html se le puede asignar un atributo para indicar en qué idioma está escrito el documento (lang) y podemos asignarle como valor un código de idioma ("es" para el español, "en" para el ingles...)
